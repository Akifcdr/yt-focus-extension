# 🎯 YT Focus Mode

A lightweight Chrome extension that blacks out YouTube distractions and shows **only your search results**, no recommendations, no homepage feed, no rabbit holes.

---

## What it blocks

| Element | Blocked |
|---|---|
| Homepage recommendations | ✅ |
| Sidebar "Up Next" suggestions | ✅ |
| End screen cards | ✅ |
| Autoplay next video | ✅ |
| YouTube Shorts shelf | ✅ |
| Homepage filter chips | ✅ |
| Masthead ads | ✅ |

Search results and your subscriptions feed are **left untouched**.

## Installation

Since this extension is not on the Chrome Web Store, you need to load it manually. It takes less than a minute.

**Step 1 — Download**

Click the green **Code** button at the top of this page → **Download ZIP** → Unzip the file.

**Step 2 — Open Chrome Extensions**

Go to `chrome://extensions` in your browser address bar.

**Step 3 — Enable Developer Mode**

Toggle **Developer Mode** on using the switch in the top-right corner.

**Step 4 — Load the extension**

Click **Load unpacked** → navigate into the unzipped folder → select the `yt-focus` folder (the one that contains `manifest.json` directly inside it).

**Step 5 — Done!**

The extension icon will appear in your toolbar. Click it anytime to toggle Focus Mode on or off.

---

## Usage

- **Focus Mode ON** — Homepage is blacked out, sidebar recommendations are gone, video player expands to full width.
- **Focus Mode OFF** — YouTube works completely normally.
- **Search** works exactly as usual at all times.

---

## Updating

Since this is not on the Web Store, updates are not automatic. To get the latest version:

1. Come back to this page
2. Download the ZIP again
3. Go to `chrome://extensions`
4. Click the **refresh icon** on the YT Focus Mode card

---

## Privacy

This extension does **not** collect, store, or transmit any personal data. It has no analytics, no tracking, and no network requests. All it does is modify what YouTube shows in your browser.
